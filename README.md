### Day 6 (Part 1)
* **Date:** Nov 21, 2025
* **Agenda:**
  * Last Class Revision
  * User Flow = User Journey
  * Gen AI => Text, Images, UI, Code, Audio, Video [All are tokens]
* **Tasks Performed in session:**
  * EDIPT has an extra invisible step, “iterative” which includes feedback like Agile Software Development(Discussion)
  * Heat maps = Where the user spends most of the time in the website or an app(Discussion)
  * How Gen AI works? Training Data -> Neural Networks -> Patterns & Representations -> Token Prediction -> Output generation
  * Multiple tokens cannot be generated once, LLM does it token by token
  * How does AI generate UI Design? Understand prompt -> Match with patterns of UI it learned -> Predict layout -> Choose Style
  * What is AI good at? Speed, Variations, Layout suggestions, Pattern recognition, auto generate components, Brainstorming
  * What is AI not good at? Deep UX insight, understanding Emotions, Real user context, Perfect Accessibility, Taste (Not reaching upto our expectations)
